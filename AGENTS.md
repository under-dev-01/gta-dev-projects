# AGENT CONFIGURATION - GTA Dev Projects

## Agents Spécialisés

### frontend-dev
- **Rôle:** Développement frontend (HTML, CSS, JavaScript, React)
- **Labels:** frontend, ui, ux, css, html
- **Skills:** Responsive design, animations, accessibilité

### backend-dev  
- **Rôle:** Développement backend (API, base de données, serveurs)
- **Labels:** backend, api, database, server
- **Skills:** Node.js, Express, SQL, API REST

### test-dev
- **Rôle:** Tests et qualité (unit tests, integration, E2E)
- **Labels:** testing, tests, qa, bug
- **Skills:** Jest, Cypress, test automation

## Configuration

Ces agents doivent être enregistrés dans OpenClaw via l'interface ou les commandes CLI.

## Utilisation

Quand une issue est créée avec le label correspondant, l'agent spécialisé est automatiquement assigné.
